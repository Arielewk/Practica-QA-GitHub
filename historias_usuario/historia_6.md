# HU-006: Gestión de Perfil de Usuario

## Historia de Usuario

Como usuario registrado de SAFEZONE,

Quiero administrar mi información personal y configuración de cuenta,

Para mantener mis datos actualizados y mejorar mi experiencia dentro de la plataforma.

## Descripción

Esta funcionalidad permitirá a los usuarios consultar, modificar y actualizar sus datos personales, garantizando que la información almacenada sea correcta y actual.

## Criterios de Aceptación

* El usuario debe poder visualizar sus datos personales.
* Debe permitirse la modificación de información personal.
* El sistema debe validar los datos ingresados.
* Los cambios realizados deben guardarse correctamente.
* El usuario debe recibir una confirmación de actualización.
* El sistema debe proteger la información personal.
* El usuario debe poder cambiar su contraseña.
* Debe mantenerse un registro de las modificaciones realizadas.
* La información actualizada debe mostrarse inmediatamente.
* Los cambios deben conservarse después de cerrar sesión.
