# HU-001: Registro de Usuario

## Historia de Usuario

Como ciudadano interesado en contribuir a la seguridad de mi comunidad,

Quiero registrarme en la plataforma SAFEZONE utilizando mis datos personales y credenciales de acceso,

Para poder acceder a todas las funcionalidades de la aplicación y participar activamente en el reporte y consulta de incidentes de seguridad.

## Descripción

La funcionalidad de registro permitirá a los ciudadanos crear una cuenta dentro de la plataforma. Esto garantizará que cada usuario sea identificado de manera única y que los reportes realizados puedan asociarse correctamente a una cuenta registrada.

## Criterios de Aceptación

* El sistema debe solicitar nombre, apellido, correo electrónico y contraseña.
* Todos los campos deben ser obligatorios.
* El correo electrónico debe tener un formato válido.
* El sistema debe verificar que el correo no exista previamente.
* La contraseña debe contener al menos ocho caracteres.
* La contraseña debe incluir letras y números.
* El sistema debe almacenar los datos correctamente.
* El sistema debe mostrar un mensaje de registro exitoso.
* El usuario debe poder acceder posteriormente con las credenciales registradas.
* En caso de error, el sistema debe informar claramente el problema.
