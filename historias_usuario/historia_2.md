# HU-002: Inicio de Sesión

## Historia de Usuario

Como usuario registrado de SAFEZONE,

Quiero iniciar sesión utilizando mi correo electrónico y contraseña,

Para acceder de forma segura a mi cuenta y utilizar las funcionalidades de la plataforma.

## Descripción

El inicio de sesión permitirá autenticar a los usuarios registrados y proteger el acceso a la información y funcionalidades disponibles dentro del sistema.

## Criterios de Aceptación

* El sistema debe solicitar correo electrónico y contraseña.
* Las credenciales ingresadas deben validarse contra la base de datos.
* El usuario debe acceder únicamente si las credenciales son correctas.
* El sistema debe mostrar mensajes de error ante datos incorrectos.
* El usuario debe poder cerrar sesión cuando lo desee.
* El sistema debe mantener la sesión activa mientras el usuario utilice la aplicación.
* Debe protegerse el acceso a funcionalidades restringidas.
* El sistema debe registrar el acceso exitoso del usuario.
* El usuario debe ser redirigido a la pantalla principal después del inicio de sesión.
* Debe existir una opción para recuperar la contraseña.
