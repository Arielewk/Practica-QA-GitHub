Nombre del Proyecto

SAFEZONE

Descripción

SAFEZONE es una aplicación orientada a la seguridad ciudadana que permite a los miembros de una comunidad reportar incidentes delictivos, consultar alertas de seguridad y conocer las zonas de riesgo mediante información compartida por los usuarios.

Funcionalidades Principales

Registro de usuario: crear una cuenta mediante nombre, correo electrónico y contraseña para acceder a la plataforma.

Inicio de sesión: acceso seguro mediante validación de credenciales registradas.

Reportar incidentes: registrar delitos, actividades sospechosas o situaciones de riesgo indicando ubicación y descripción del evento.

Consultar alertas de seguridad: visualizar los reportes realizados por otros usuarios para mantenerse informado sobre incidentes recientes.

Visualizar mapa de incidentes: observar en un mapa interactivo las zonas donde se han reportado eventos de inseguridad.

Gestión de perfil: actualizar información personal y administrar los datos de la cuenta.
