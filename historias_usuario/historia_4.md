# HU-004: Consultar Alertas de Seguridad

## Historia de Usuario

Como usuario de SAFEZONE,

Quiero visualizar las alertas de seguridad registradas por otros usuarios,

Para mantenerme informado sobre los riesgos existentes en mi comunidad.

## Descripción

La plataforma debe ofrecer una sección donde se muestren los reportes e incidentes recientes, permitiendo a los ciudadanos conocer situaciones que puedan afectar su seguridad.

## Criterios de Aceptación

* El sistema debe mostrar una lista de alertas activas.
* Cada alerta debe incluir fecha, ubicación y descripción.
* Las alertas más recientes deben aparecer primero.
* El usuario debe poder consultar información detallada.
* El sistema debe actualizar la información periódicamente.
* Debe permitirse filtrar alertas por categoría.
* Debe permitirse buscar alertas por ubicación.
* La información mostrada debe ser clara y comprensible.
* El sistema debe garantizar la disponibilidad de los datos.
* Las alertas deben estar visibles para todos los usuarios autorizados.
