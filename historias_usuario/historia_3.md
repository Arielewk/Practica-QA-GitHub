# HU-003: Reportar Incidente de Seguridad

## Historia de Usuario

Como usuario registrado,

Quiero reportar incidentes de seguridad ocurridos en mi comunidad,

Para informar a otros ciudadanos y ayudar a prevenir situaciones de riesgo.

## Descripción

Esta funcionalidad permitirá a los usuarios registrar eventos relacionados con robos, asaltos, vandalismo, actividades sospechosas u otros incidentes que afecten la seguridad de la comunidad.

## Criterios de Aceptación

* El usuario debe estar autenticado para crear un reporte.
* El sistema debe permitir seleccionar el tipo de incidente.
* Debe existir un campo para describir detalladamente lo ocurrido.
* El usuario debe poder indicar la ubicación del incidente.
* El sistema debe registrar fecha y hora del reporte.
* El usuario debe poder adjuntar evidencia si está disponible.
* El sistema debe validar los campos obligatorios.
* El reporte debe almacenarse correctamente.
* El sistema debe confirmar el envío exitoso.
* Otros usuarios deben poder visualizar el reporte publicado.
