# HU-005: Visualizar Mapa de Incidentes

## Historia de Usuario

Como usuario de SAFEZONE,

Quiero consultar un mapa interactivo con los incidentes reportados,

Para identificar las zonas con mayor nivel de riesgo y tomar precauciones.

## Descripción

La aplicación debe representar gráficamente los incidentes registrados mediante marcadores geográficos que permitan localizar rápidamente las áreas afectadas.

## Criterios de Aceptación

* El sistema debe mostrar un mapa interactivo.
* Cada incidente debe representarse mediante un marcador.
* Los marcadores deben mostrar información al seleccionarlos.
* El mapa debe reflejar la ubicación real del incidente.
* Deben visualizarse los incidentes más recientes.
* El sistema debe actualizar automáticamente los nuevos reportes.
* El usuario debe poder acercar y alejar el mapa.
* Debe ser posible filtrar incidentes por categoría.
* El mapa debe funcionar correctamente en dispositivos móviles.
* La información geográfica debe ser precisa y consistente.
